# AI吵架生成器 — 用户流程图

## 1. 整体用户流程

```
                    ┌─────────────┐
                    │   打开小程序   │
                    └──────┬──────┘
                           │
                    ┌──────▼──────┐
                    │   首页(Tab1)  │
                    │  输入场景     │
                    │  选择风格     │
                    └──┬───┬───┬──┘
                       │   │   │
          ┌────────────┘   │   └────────────┐
          ▼                ▼                ▼
   ┌──────────┐   ┌──────────────┐   ┌──────────┐
   │历史(Tab2) │   │ 点击「生成」  │   │我的(Tab3)│
   │查看历史   │   └──────┬───────┘   │查看收藏   │
   │点击某条   │          │           │查看次数   │
   └──┬───────┘          ▼           └──────────┘
      │          ┌──────────────┐
      │          │   结果页      │
      │          │  展示文案     │
      │          └──┬───┬───┬──┬┘
      │             │   │   │  │
      │    ┌────────┘   │   │  └───────────┐
      │    ▼            ▼   ▼              ▼
      │  ┌─────┐  ┌─────┐ ┌─────┐  ┌──────────┐
      └──►│复制 │  │换风格│ │再来 │  │分享/收藏 │
          └─────┘  └─────┘ │一句 │  └──────────┘
                           └─────┘
```

## 2. 首页输入流程

```
用户打开首页
      │
      ▼
┌──────────────────┐
│ 方式一：手动输入   │────► 在文本框输入场景描述
└──────────────────┘       （例："同事甩锅给我"）
      │
      ▼
┌──────────────────┐
│ 方式二：预设场景   │────► 点击预设场景标签
└──────────────────┘       （例：同事甩锅 | 对象冷暴力 | 亲戚催婚）
      │
      ▼
┌──────────────────┐
│ 选择生成风格      │────► 单选风格
└──────────────────┘       （默认选"发疯文学版"）
      │
      ▼
┌──────────────────┐
│ 点击「生成怼人话」 │
└──────┬───────────┘
       │
       ▼
     结果页
```

## 3. 结果页操作流程

```
进入结果页（展示生成文案）
      │
      ├──► 点击「复制」→ Toast提示"已复制" → 留在当前页
      │
      ├──► 点击风格Tab（高情商/阴阳怪气/发疯/文艺/霸总）
      │       └──► 请求同场景新风格内容 → 更新展示
      │
      ├──► 点击「再来一句」
      │       └──► 同场景同风格重新生成 → 更新展示
      │
      ├──► 点击「收藏」
      │       └──► 收藏/取消收藏 → 更新收藏状态
      │
      └──► 点击「生成图片」
              └──► 生成分享卡片 → 长按保存/分享
```

## 4. 历史页流程

```
进入历史页
      │
      ▼
┌──────────────────┐
│ 时间倒序列表      │
│ 每条显示：        │
│ - 场景描述        │
│ - 风格标签        │
│ - 生成时间        │
│ - 文案预览(2行)   │
└──────┬───────────┘
       │
       ├──► 点击某条 → 跳转结果页（展示该条完整内容）
       │
       └──► 左滑某条 → 删除按钮 → 确认删除
```

```
进入我的页
      │
      ▼
┌──────────────────┐
│ 使用统计          │
│ - 今日生成次数    │

      │
      ▼

│ 我的收藏          │
│ 列表展示收藏内容   │
│ 点击可查看详情    │
└──────────────────┘
```
