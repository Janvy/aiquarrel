# AI吵架生成器 — 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品名称

AI吵架生成器

### 1.2 产品定位

轻娱乐 · 情绪表达 · 社交传播

一款帮助用户用幽默、高情商方式表达情绪的 AI 生成工具。核心不是教人吵架，而是把负面情绪转化为可分享的趣味内容。

### 1.3 目标用户

| 维度 | 描述 |
|------|------|
| 年龄 | 18~35 岁 |
| 特征 | 互联网原住民，熟悉表情包/发疯文学等网络文化 |
| 场景 | 职场社交、亲密关系、家庭关系中的情绪表达需求 |
| 动机 | 想怼回去但不想真的破坏关系；想发泄但要有趣不低级 |

### 1.4 核心价值

- **情绪出口**：把憋屈转化为有趣的文字
- **社交货币**：生成的内容天生适合分享到朋友圈/群聊
- **零门槛**：不需要会吵架，不需要会写文案

|------|------|----------|
| S1 | 同事甩锅 | "这个明明是你的部分，现在说我没交接" |

| S6 | 朋友借钱不还 | "就两千块钱至于天天催吗" |

## 3. 生成风格

| 编号 | 风格 | 调性描述 | 示例（场景：室友半夜外放抖音） |
|------|------|----------|------|
| T1 | 高情商版 | 温和但坚定，不伤和气 | "我睡眠比较浅，过了11点有点声音就睡不着，咱能商量一下吗" |
| T2 | 阴阳怪气版 | 讽刺但不带脏字，绵里藏针 | "原来抖音还有外放功能，我一直以为只能戴耳机呢，涨知识了" |
| T3 | 发疯文学版 | 荒诞夸张，情绪拉满 | "哈哈我不活了谁懂啊半夜两点全世界都在震动地球是不是要爆炸了" |
| T4 | 文艺版 | 文艺优美，伤感到位 | "夜深了，我的梦却被你的抖音偷走了" |
| T5 | 霸总版 | 霸道强势，反差萌 | "女人/男人，给你三秒钟关掉那个声音，否则后果自负" |

## 4. V1 功能范围

### 4.1 功能清单

| 模块 | 功能 | 优先级 | 说明 |
|------|------|--------|------|
| 首页 | 场景输入 | P0 | 文本框输入 + 预设场景快捷选择 |
| 首页 | 风格选择 | P0 | 5种风格单选，默认随机 |
| 首页 | 一键生成 | P0 | 点击按钮，调用AI生成 |
| 结果页 | 展示结果 | P0 | 展示单条生成结果 |
| 结果页 | 风格切换 | P0 | Tab切换查看同一场景的不同风格 |
| 结果页 | 一键复制 | P0 | 复制文案到剪贴板 |
| 结果页 | 再来一句 | P0 | 同场景同风格重新生成 |
| 结果页 | 分享图片 | P1 | 生成精美卡片图，保存/分享 |
| 历史页 | 历史记录列表 | P0 | 按时间倒序展示 |
| 历史页 | 清除历史 | P2 | 单条删除 + 全部清除 |
| 我的 | 使用次数 | P1 | 显示当日/累计使用次数 |
| 我的 | 收藏列表 | P0 | 查看已收藏的生成结果 |
| 结果页 | 收藏 | P0 | 收藏当前结果 |

### 4.2 暂不纳入 V1

- 用户登录/注册（先匿名使用）
- 分享到朋友圈（仅支持生成图片保存）
- 语音输入
- 社区/广场（UGC 内容浏览）
- 自定义风格

---

## 5. 非功能需求

| 维度 | 要求 |
|------|------|
| 生成速度 | 用户点击后 3 秒内出结果 |
| 内容安全 | 不生成人身攻击、歧视性、违法内容 |
| 并发 | 支持 100 QPS |
| 平台 | 微信小程序（iOS + Android） |
| 审核 | 符合微信小程序内容审核规范 |

---

## 6. 风险与应对

| 风险 | 影响 | 应对 |
|------|------|------|
| 被判定为传播负能量 | 审核不通过 | 弱化"吵架"字眼，强调"情绪表达""幽默化解" |
| AI 生成低俗/违规内容 | 封禁风险 | Prompt 增加安全约束 + 敏感词过滤 |
| 用户下沉后内容同质化 | 留存下降 | 预设场景引导 + 丰富风格模板 |
| 微信小程序审核 | 上线受阻 | 选择"工具-效率"类目，避免"社交"类目 |
